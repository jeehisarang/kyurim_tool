# 작업지시서 — 검사톡/공유링크 섹션 표시 순서 변경

## 배경
현재 프로그램티칭+이벤트+검사결과를 함께 묶은 공유링크(/s/[token])에서
섹션 노출 순서가 "프로그램티칭 → 이벤트 → 검사결과" 순으로 되어 있음.
원장님 확인 결과 순서를 아래로 변경 요청.

## 변경 규칙
표시 순서를 항상 다음으로 고정:
1. 검사결과 (검사톡)
2. 프로그램티칭
3. 이벤트 (항상 맨 마지막)

포함된 조합이 무엇이든(2개만 있어도, 3개 다 있어도) 위 순서 규칙을
동일하게 적용할 것. 예: 티칭+이벤트만 있는 기존 링크는 "프로그램티칭 →
이벤트" 순서 유지(검사결과 없으니 그대로), 검사+이벤트만 있으면 "검사결과
→ 이벤트" 순서.

## 대상
- /s/[token] 공개 페이지 렌더링 순서
- (관리자 쪽 "링크 포함하기" 미리보기 화면도 동일 순서로 보이면 함께 맞출 것,
  단 체크박스 UI 배치 자체를 바꾸라는 건 아니고 실제 표시 콘텐츠 순서만)

## 검증
- 테스트 환자(426)로 3종 조합(검사+티칭+이벤트) 링크 재생성 후 순서 확인
- 기존에 이미 생성된 실제 환자 링크는 재생성 전까지 기존 순서로 남아있을
  텐데, 이건 문제 없음(신규 생성분부터 적용이면 충분)
