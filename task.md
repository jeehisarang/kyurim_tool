# 작업지시서 — HRV 상세지표 해석 최종 구현 (자율신경균형도 앵커 + 이미지 1장 전환)

## 1. 배경 및 확정된 결론

지난 조사 2건 결과, 아래가 확정됐습니다:

1. 표준범위 계산 공식은 못 찾았고, 찾을 필요가 없다 — 업로드되는 종합
   리포트 이미지 안에 이미 "초록 막대(표준범위) + 빨간 마커(실측값)"로
   TP/VLF/LF/HF/LF-HF/평균맥박/SDNN/RMSSD 각각 시각화되어 있음. AI가 이
   이미지를 그대로 읽어서 언급하면 되므로, 별도 기준표/공식 확보 불필요
2. 업로드 이미지는 이미 종합 리포트 1장 — 자율신경균형도 매트릭스,
   맥박다양성(+기기 해석 문구), 상세분석 막대그래프가 전부 한 이미지에
   포함되어 있음이 실제 환자 레코드로 확인됨
3. 운영규칙이 "스트레스 지수 측정 1회만 진행"으로 바뀌었으므로, 앞으로는
   항상 이 종합 리포트 1장만 나옴 (2페이지로 나뉠 일이 없음)

이에 따라 아래 2가지를 함께 구현합니다.

## 2. 작업 A — 이미지 업로드를 1장으로 축소

- 현재 HRV 등록 화면의 2페이지 업로드 슬롯을 1장으로 변경
- 기존에 이미 2장으로 저장된 과거 레코드는 그대로 유지 (마이그레이션 불필요,
  단순히 신규 등록 시 슬롯 UI만 1개로 축소)
- 관련 레이아웃(좌 이미지/우 수치입력 2단 배치 등 지난 세션에 만든 UI)은
  이미지 슬롯 개수 변경에 맞춰 자연스럽게 조정

## 3. 작업 B — AI 코멘트에 자율신경균형도 앵커 반영

기존 "결과와 추이" / "한의학적 해석" 섹션 로직을 확장합니다 (새 섹션을
추가하는 게 아니라 기존 4슬롯 구조 안에서 내용만 강화):

- 결과와 추이 섹션: 기존 4개 지표(혈관건강지수/등급/스트레스지수/
  평균맥박) 추이 서술에 더해, 이미지에서 읽은 자율신경균형도 유형
  (과로형만성스트레스/질병형만성스트레스/급성스트레스/초기부정맥/
  심한부정맥 중 어디에 위치하는지)과 맥박다양성 값을 함께 서술. 지난
  기록이 있는 경우 유형이 어느 방향으로 이동했는지도 추이로 언급
  (기존 "2회차 이상부터 추이 언급" 규칙 그대로 적용)
- 한의학적 해석 섹션: 자율신경균형도 유형을 미병 신호로 재서술하고,
  기존 패턴명 매칭 로직(증상 텍스트 있을 때만 언급)과 자연스럽게 연결
- 세부지표(TP/VLF/LF/HF/SDNN/RMSSD) 개별 판정은 나열하지 않음 — 위
  두 앵커(자율신경균형도, 맥박다양성)를 중심으로 서술하고, 세부지표는
  필요한 경우에만 보조적으로 1~2개 언급하는 정도로 제한 (전체 나열 금지 —
  나열하면 다시 "재탕"이 됨)

### 안전장치 (반드시 유지)
- AI가 이미지에서 값을 읽을 때, 막대/마커가 흐릿하거나 잘려서 명확히
  안 보이면 추측해서 서술하지 말고 언급을 생략하도록 프롬프트에 명시
  (수치 조작·환각 방지)
- 기존 안전원칙(패턴명은 증상 텍스트 매칭 시에만, 수치만으로 추론 금지,
  "최종 변증은 문진·설진·맥진으로 확정" 유보 문구) 그대로 유지
- 지난 세션에 추가한 "패턴명 오탐지 시 재시도 → 실패시 코멘트 실패 처리"
  안전장치도 이번 확장된 프롬프트에 동일 적용

## 4. 작업 C — CSV 자동임포트 경로의 AI 코멘트 처리 확인/보완

지난 완료보고에서 답변이 빠졌던 부분입니다. CSV로 자동 생성되는 HRV
기록에도 AI 코멘트가 함께 자동 생성되는지 확인하고, 만약 아니라면 자동
생성되도록 보완해주세요. 이때:
- 수동 등록 경로와 동일한 안전장치(오탐지 재시도/실패처리)가 자동임포트
  경로에도 동일하게 적용되어야 함
- 자동임포트로 생성된 기록의 AI 코멘트 생성이 실패한 경우, 미매칭
  대기열과는 별도로 "코멘트 생성 실패" 표시가 원장 확인화면에 보이는지
  확인 (조용히 묻히면 안 됨)

## 5. 검증 체크리스트

- 테스트 환자(426)로 종합 리포트 이미지 1장만 업로드해서 등록 → 자율신경
  균형도 유형과 맥박다양성이 코멘트에 반영되는지 확인
- 2회차 등록 시 유형 이동 방향이 추이로 서술되는지 확인
- 이미지 일부가 잘려서 안 보이는 케이스(예: 일부러 자른 이미지)로 테스트해서
  AI가 추측하지 않고 자연스럽게 생략하는지 확인
- 증상 텍스트 없는 케이스에서 패턴명 여전히 언급 안 되는지 회귀 확인
- CSV 자동임포트 경로에서도 AI 코멘트가 정상 생성/실패처리되는지 확인
- 기존 2페이지로 저장된 과거 레코드 조회 시 문제없는지 확인
- 실데이터 자동화 스크립트 금지, 테스트 환자로만 검증
