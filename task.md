## 작업: /s/[token] 이벤트 섹션에 카톡 본문(설명문구) 추가 표시

### 배경
- 현재 /s/[token] 페이지의 이벤트 섹션은 이미지(제목+인트로 합성본)만 표시하고,
  EventImage의 카톡 본문 전체 텍스트(copy 필드)는 표시하지 않고 있음
- 원장님 확인 결과, 이 설명문구도 환자가 링크를 눌렀을 때 이미지와 함께 보여야
  자연스럽다는 판단 — 이미지 아래에 카톡 본문 텍스트도 함께 렌더링할 것

### 수정 사항
- /s/[token] 공개 페이지의 이벤트 섹션: 기존 이미지(title+intro 합성본) 아래에
  EventImage.copy(카톡 본문 전체 텍스트) 를 일반 텍스트 블록으로 추가 표시
  - 줄바꿈(문단 구분) 유지해서 렌더링 (톡생성기 화면의 "카카오톡 본문" textarea에
    보이는 것과 동일한 내용/줄바꿈으로)
  - 디자인은 기존 티칭지 섹션의 본문 블록 스타일(카드형 텍스트박스 등)과 톤 맞춰서
    자연스럽게
- "완성된 이미지 그대로 사용" 모드(Canvas 합성 없이 원본 이미지 그대로 저장한
  경우)에도 동일하게 copy 텍스트는 그 아래에 표시되어야 함
- 통합(티칭지+이벤트) 페이지에서도 동일하게 적용 (이벤트 섹션 자체에 포함되는
  부분이므로 순서상 티칭지 다음, 이벤트 이미지+본문 순으로 표시)

### 검증 요청
1. 이벤트만 포함된 /s/[token] → 이미지 아래 카톡 본문 전체가 줄바꿈 유지된 채
   정상 표시되는지 확인
2. 통합(티칭지+이벤트) → 티칭지 섹션 다음 이벤트 섹션(이미지+본문) 순서로 정상
   표시되는지 확인
3. "완성된 이미지 그대로 사용" 모드로 저장된 이벤트로도 동일하게 본문 텍스트
   나오는지 확인
4. npx tsc --noEmit, npm run build 통과 확인
