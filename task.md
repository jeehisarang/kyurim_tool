## 동작 규칙
1. `.env`에 다음 값 설정:
   HRV_CSV_FOLDER_PATH=G:\내 드라이브\규림한의원천안점(공유용)\맥파기검사기록지\csv
   - Windows 경로 백슬래시 이스케이프 주의 (Node.js에서 경로 처리 시 path.join 등으로 
     정규화해 하드코딩 이스케이프 문제 방지)
2. 폴링 주기는 기존 구현값 유지 (변경 필요 시 별도 논의)
3. 매칭 성공 시 즉시 실제 DB(HrvTestRecord)에 저장 — 시뮬레이션/드라이런 모드 없이 라이브로 동작
4. 차트번호 컬럼이 비어있거나 매칭 실패 시 기존 로직대로 HrvImportPending(미매칭 대기열)에 적재
5. 폴더 접근 실패(드라이브 마운트 해제 등) 시 서버 크래시 없이 로그만 남기고 다음 폴링 주기에 재시도
