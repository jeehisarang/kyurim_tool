# 작업지시서 — 카테고리 색상 고정매핑 + 치료방향 카드 색상/아이콘/한자 병기

## 배경
- 카드4(한의건강해석) 상단 카테고리 비중 시각화(도넛+막대 조합)와, 카드7(치료 방향)의
  카테고리별 카드가 서로 색상으로 매칭되면 상담 중 시각적 연결이 쉬워진다는 원장 피드백.
- 변증명(소간해울 등)에 한자 병기 + 볼드 처리 요청.
- 카테고리별 대표 아이콘 추가 요청.

## 사전확인 체크리스트 (설계 승인 완료)

### 1) 카테고리 고정 색상 매핑 (7개 전체, 등장 순서 아닌 카테고리 고유 색)
| 카테고리 | 색상 |
|---|---|
| 스트레스·정서긴장 | 파랑 #2a78d6 |
| 기력·냉증 | 주황 #eb6834 |
| 열감·건조 | 청록 #1baf7a |
| 소화기 | 노랑 #eda100 |
| 담습·부종 | 분홍 #e87ba4 |
| 혈허 경향 | 초록 #008300 |
| 순환·어혈 | 보라 #4a3aa7 |
- 이 매핑을 코드 상수(`TCM_CATEGORY_COLOR_MAP`)로 고정, 카드4 시각화와 카드7
  둘 다 이 상수를 참조 (색상이 두 군데서 따로 관리되지 않도록)

### 2) 카드4 — 카테고리 비중 시각화, 도넛(좌)+막대(우) 조합
- 계산 방식은 기존 확정안 유지: 전체 응답 문항 만점 대비 원점수 비율(정규화 없음)
- 도넛: 후보 카테고리들 + "기타"(후보 아닌 나머지 비중) 한 조각으로 표시
- 막대: 후보 카테고리만, 각각 라벨+퍼센트
- 두 시각화 모두 위 고정 색상 매핑 사용

### 3) 카드7 — 치료방향 카드 개선 (색상/아이콘/한자)
- 카드 좌측 3px 컬러 바 — 해당 카테고리 고정 색상
- 카드 제목 옆 아이콘 (카테고리당 고정 1개, 아래 초안 — 실제 적용 후 스크린샷으로
  원장 확인, 마음에 안 들면 교체):
  - 스트레스·정서긴장: mood-sad
  - 기력·냉증: temperature-minus
  - 열감·건조: flame
  - 소화기: stomach
  - 담습·부종: droplet
  - 혈허 경향: heartbeat (아이콘명 실존 확인 필요, 없으면 대체)
  - 순환·어혈: activity
  - 생활관리(공통): heart, 색상은 중립 회색
- 변증명(소간해울 등) 볼드 처리 + 괄호 안 한자 작은 회색 글씨로 병기
- 한자는 AI가 생성하지 않고 **고정 사전**(키워드-한자 매핑)에서 후처리로 삽입
  — 아래 28개 매핑 그대로 코드에 반영. **단, 한자 표기 정확도는 원장님이
  최종 검수 후 확정** (제가 정리한 것이나 한의학 전문 검수 권장)

## 고정 사전 — 키워드·한자 매핑 (원장 검수 후 확정)

스트레스·정서긴장
- 소간해울(疏肝解鬱), 이기해울(理氣解鬱), 청간사화(淸肝瀉火), 양심안신(養心安神)

기력·냉증
- 보기익기(補氣益氣), 건비익기(健脾益氣), 온양산한(溫陽散寒), 온보비신(溫補脾腎)

열감·건조
- 자음생진(滋陰生津), 자음청열(滋陰淸熱), 자음강화(滋陰降火), 보익간신(補益肝腎)

소화기
- 건비화위(健脾和胃), 이기화중(理氣和中), 소도화적(消導化積), 화담강역(化痰降逆)

담습·부종
- 건비화습(健脾化濕), 화담이수(化痰利水), 이수소종(利水消腫), 온양화수(溫陽化水)

혈허 경향
- 보혈양혈(補血養血), 익기생혈(益氣生血), 양심안신(養心安神), 건비생혈(健脾生血)

순환·어혈
- 활혈거어(活血祛瘀), 행기활혈(行氣活血), 통락지통(通絡止痛), 온경산한(溫經散寒)

## 동작 규칙
- AI가 생성한 문장 안에서 위 28개 키워드 문자열을 찾아, 코드가 `**키워드**(한자)` 형태로
  후처리 삽입 (AI 출력 자체는 그대로, 후처리로 볼드+한자만 추가)
- 키워드가 여러 번 등장해도(예: 양심안신이 혈허경향/스트레스 양쪽에 있음) 카테고리
  문맥과 무관하게 항상 동일한 한자 삽입
- 사전에 없는 키워드가 등장하면(향후 확장 대비) 볼드만 처리, 한자는 생략 + 콘솔 경고
