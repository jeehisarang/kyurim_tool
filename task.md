## 작업: 링크 자동첨부 시 안내문구 추가 (프로그램티칭/이벤트/통합 3종)

### 배경
- 현재 톡생성기에서 "링크 포함하기"로 프로그램티칭/이벤트/통합을 선택하고 링크를
  생성하면, 메인 톡 문구 복사 시 URL이 자동으로 하단에 붙어서 함께 복사됨
- 지금은 URL만 그냥 붙는데, 그 앞에 짧은 안내 문장이 함께 나오도록 개선

### 추가할 고정 템플릿 (환자 이름만 치환, AI 호출 없음)
- 프로그램티칭만: `{환자이름}님의 검사 결과와 추천 프로그램을 아래 링크에서 확인해보세요 🙂`
- 이벤트만: `{환자이름}님을 위한 특별한 혜택을 아래 링크에서 확인해보세요 🙂`
- 통합(티칭지+이벤트): `{환자이름}님의 검사 결과와 추천 혜택을 아래 링크에서 확인해보세요 🙂`

### 최종 복사 형태 (예시)
최승희님의 검사 결과와 추천 혜택을 아래 링크에서 확인해보세요 🙂
http://192.168.0.121:3001/s/wcz2jeUW

### 수정 사항
- 링크 자동첨부 로직(현재 URL만 하단에 붙이는 부분)을 찾아서, 선택된 조합
  (프로그램티칭/이벤트/통합)에 따라 위 3종 템플릿 문장 + 줄바꿈 + URL 형태로 교체
- 환자 이름은 이미 화면에서 선택된 환자 정보로 채워져 있으니 그대로 활용
- 기존 "링크만 복사" 버튼(URL만 단독 복사)은 그대로 유지 — 이건 안내문구 없이
  URL만 필요할 때를 위한 것이므로 변경 없음
- 메인 톡 문구(내원톡/2일톡/7일톡/3회차톡 등) + 안내문구 + URL이 합쳐질 때
  문구와 안내문구 사이에 빈 줄 하나 넣어서 시각적으로 구분되게 할 것

### 검증 요청
1. 프로그램티칭만 선택 → 복사 결과에 해당 템플릿 문구 + URL 정상 포함 확인
2. 이벤트만 선택 → 해당 템플릿 문구 + URL 정상 포함 확인
3. 통합 선택 → 해당 템플릿 문구 + URL 정상 포함 확인
4. "링크만 복사" 버튼은 여전히 URL만 복사되는지(회귀 없음) 확인
5. 환자 이름이 실제로 정확히 치환되는지 확인
6. npx tsc --noEmit, npm run build 통과 확인
