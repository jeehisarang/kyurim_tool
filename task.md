# 작업 지시서 — 신청 완료 화면에 카카오채널 대응 안내문구 추가

## 배경
신청폼("신청하기") 제출 시 카카오채널 채팅창이 자동으로 열리지만, 사용자가
아무 액션도 하지 않으면 원장님 쪽 "내 채팅"에 상담 건으로 잡히지 않아 연락할
방법이 전화/문자밖에 없어지는 문제가 있음. 알림톡 자동발송(별도 준비 진행 중)이
붙기 전까지, 최소한 사용자에게 "여기서 뭘 해야 하는지" 고지가 필요함.

확인된 사실: "관리자에게 문의하기" 클릭 / 리스트 메뉴("킬팻캡슐 3일 체험 신청")
클릭 / 채팅창에 직접 메시지 입력 — 이 셋 중 아무거나 하나만 해도 상담 건으로
등록됨. 순서대로 다 거쳐야 하는 게 아니므로, "여러 액션 중 하나만 하면 됨"으로
안내할 것.

## 수정
- `TrialApplicationForm.tsx`의 완료 화면(제출 성공 후 화면)에 아래 안내문구
  추가:

카카오톡 채팅창이 열립니다.

아래 중 아무거나 한 가지만 해주세요:

"관리자에게 문의하기" 누르기
메뉴에 뜨는 "킬팻캡슐 3일 체험 신청" 누르기
채팅창에 직접 아무 메시지나 입력하기

이 중 하나만 하시면 저희가 신청 내용을 확인하고 연락드릴 수 있어요!

- 문구는 카카오 채팅창이 열리는 동작(`window.open`) 직후, 완료 화면 상단에
  눈에 띄게 배치 (기존 "신청 완료되었습니다" 문구 아래)
- 마감설문(`ExitSurveyForm.tsx`)은 카카오채널 연결 기능 자체가 없으므로 이번
  수정 대상 아님

## 검증
- 신청 제출 후 완료 화면에 안내문구가 정상 노출되는지 확인
- 실제 카카오채널과 대화 이력이 없는 새 계정으로 전체 흐름(신청→채팅오픈→
  안내문구 확인→아무 액션 하나 실행→내채팅 확인)까지 재현 테스트
- npx tsc --noEmit / npm run build 통과
