# 작업지시서 — HRV AI 코멘트 "미병(未病)" 프레임 재설계 (1단계: HRV만)

## 1. 배경 및 목적

현재 HRV AI 코멘트는 기기가 계산한 수치/등급을 문장으로 풀어쓰는 수준에 머물러
있어, 원장님 표현으로 "기존 검사기록지 재탕"에 가깝다는 문제의식이 있음.

핵심 방향 전환: 한의학의 "미병(未病)"/"치미병(治未病)" 개념 — 질병으로
확정되기 전 몸의 불균형 신호를 미리 포착해 예방적으로 관리한다는 개념 — 을
HRV 코멘트의 중심 철학으로 삼는다. 이는 양방의 "증상에 대한 대증치료"와
대비되는 한의학 고유의 장점이며, 원장님이 진료 전반에서 강조하고 싶어하는
가치다.

미병 개념은 본질적으로 "지금 이 순간의 수치가 정상/비정상인가"가 아니라
"이 사람의 몸이 어느 방향으로 흘러가고 있는가"를 보는 것이므로, 추이(변화
방향) 비교를 코멘트의 핵심 요소로 넣는 것이 미병 프레임 구현의 필수 조건임.

이번 작업은 HRV 검사 코멘트에만 적용한다 (인바디/근력검사, 프로그램티칭
문구 확장은 이번 범위 아님, 추후 별도 세션에서 진행 예정이므로 건드리지 말 것).

## 2. 착수 전 확인 필요 (설계 먼저 제시)

구현 전 아래를 코드에서 확인하고 계획을 정리해서 먼저 보고할 것 (승인 후
실제 구현 진행):

1. 현재 HRV 코멘트 생성 프롬프트 위치/구조 확인
   - 기기판독요약/임상적의미/생활관리안내/한의학적해석/안전안내 5개 섹션이
     실제로 어느 파일(프롬프트 템플릿)에서 어떻게 구성되는지 확인
   - 안전안내가 "시스템이 고정 추가"하는 방식이라는 기존 원칙이 실제 코드에서
     어떻게 구현돼 있는지 확인 (이번 작업에서 안전안내는 그대로 유지)

2. 같은 환자의 과거 HRV 기록 조회 방법 확인
   - HRV 등록/코멘트 생성 시점에 해당 환자의 과거 HRV 기록(날짜순)을 조회하는
     기존 로직이 있는지 확인 (있다면 "지난 검사 보기"류 컴포넌트에서 재사용)
   - 없다면 신규로 조회 쿼리를 추가해야 하는데, 이 경우 조회 범위(최근 몇
     건까지 가져올지 — 최소 직전 1건은 필수, 추이 서술을 더 풍부하게 하려면
     최근 3건 정도까지 가져오는 안을 제안)

3. 안전원칙 재확인 (기존 원칙과 충돌 없는지)
   - 기존 규칙: "한의학 변증명은 증상 텍스트가 매핑 테이블과 실제로 일치하는
     경우에만 언급, 수치만으로 추론 금지" — 이 원칙은 이번 재설계에서도 절대
     유지되어야 함. 미병 프레임으로 톤을 바꾸는 것이지, 패턴명 언급 조건을
     완화하는 것이 아님을 프롬프트에 명확히 유지할 것
   - "최종적인 변증은 문진·설진·맥진을 통해 확정됩니다" 류의 유보 문구도 계속
     유지

## 3. 신규 코멘트 구조 (설계 승인 후 적용)

기존 5섹션 구조를 아래처럼 재구성한다 (섹션 개수/이름은 유지하되 내용과
톤을 재설계):

### 섹션 1. 미병 도입 (신규 추가, 맨 앞)
- AI가 이번 측정 수치에 맞춰 매번 다르게 생성 (고정 문구 아님, 원장님 결정)
- 핵심 메시지: "이 검사는 질병 여부를 가르는 것이 아니라, 아직 병으로
  나타나지 않은 몸의 불균형 신호(미병)를 조기에 살펴보는 것"이라는 취지를
  이번 수치 맥락에 맞게 자연스럽게 풀어씀
- 진단처럼 들리지 않도록 주의 — "~일 수 있습니다", "~신호로 볼 수 있습니다"
  같은 유보적 어조 유지

### 섹션 2. 이번 결과 + 추이 (기존 "기기판독요약" 확장)
- 과거 기록이 없는 첫 검사: 추이 언급 없이 현재 수치/등급 상태만 설명
  (기존 방식과 동일)
- 과거 기록이 1건 이상 있는 경우(=이번이 2회차 이상): 직전 기록과 비교해
  방향성(호전/악화/유지)을 언급. 단, 2개 데이터 포인트만으로 단정적으로
  말하지 않도록 "지난 검사 대비 OOO 방향으로 변화가 있었습니다" 정도의
  신중한 표현 사용 — "확실히 좋아지고 있다/나빠지고 있다" 같은 단정적 표현
  금지
- 비교 대상 지표: 혈관건강지수, 혈관건강도 등급, 스트레스지수, 평균맥박
  4개 모두 각각 방향성 언급

### 섹션 3. 한의학적 해석 (기존 유지 + 톤 재구성)
- 패턴명 언급 조건은 기존 그대로 유지 (증상 텍스트 매칭 시에만, 수치만으로
  추론 금지)
- 톤만 "진단명을 알려주는 것"에서 "미병 단계에서 나타날 수 있는 신호"로
  재구성 — 예: "간기울결 패턴 가능성을 시사합니다" → "정서적 긴장이 누적되며
  나타날 수 있는 간기울결 유형의 미병 신호로 볼 수 있습니다" 류 (정확한 문구는
  Claude Code가 프롬프트 작성 시 다듬을 것)
- "최종적인 변증은 문진·설진·맥진을 통해 확정됩니다" 유보 문구 유지

### 섹션 4. 양생 안내 (기존 "생활관리안내" 리브랜딩)
- "치미병을 위한 양생법"이라는 한의학적 틀로 재구성
- 오늘 정리한 유비오넷 자료실 콘텐츠(스트레스와 혈액순환/소화/면역 등)의
  생리학적 설명을 참고해도 좋으나, 원자료가 학술논문이 아닌 제조사 블로그
  성격이므로 "연구에 따르면"이 아니라 "임상적으로", "~하는 경우가 많습니다"
  수준의 톤 유지 (해당 정리본 파일 참고: 유비오맥파_자료실_스트레스자율신경_정리.md)
  — 약물 관련 내용(idx=289)은 절대 코멘트에 반영하지 말 것

### 섹션 5. 안전 안내 (기존 그대로 유지)
- 이번 작업과 무관, 시스템이 고정 추가하는 방식 그대로 유지

## 4. 검증 체크리스트

- 테스트 환자(426)로 HRV 기록을 최소 2건(1차, 2차) 생성 — 1차는 추이 없이,
  2차는 1차 대비 추이가 코멘트에 반영되는지 확인
- 1차 검사만 있는 경우(과거 기록 0건) 코멘트에 추이 언급이 나오지 않고
  자연스러운지 별도 확인
- 미병 도입 문구가 매번 다르게 생성되는지(고정 문구 반복 아닌지) 여러 번
  재생성해서 확인 (기존 "강제 재생성" 버튼 활용)
- 패턴명이 증상 텍스트 없는 케이스(예: 증상 기록 0건)에서 여전히 언급되지
  않는지 회귀 확인 — 이건 지난 세션 발견 이슈의 재발 방지 차원에서 반드시
  확인할 것
- 안전 안내 섹션은 이번 변경과 무관하게 그대로인지 확인
- 원장 확인화면과 환자용 화면 둘 다 새 코멘트 구조가 정상 반영되는지 확인
- 실데이터에 자동화 스크립트 실행 금지, 테스트 전용 환자로만 검증

## 1. 자율신경 기본 개념

### 스트레스와 자율신경 (idx=282)
자율신경은 오장육부·혈관·땀샘 등 모든 장기의 작용을 무의식적으로 조절하는 신경으로,
교감신경과 부교감신경이 서로 반대 방향으로 작동한다. 낮 활동 시엔 교감신경이 우위가
되어 혈관 수축·맥박 상승·호흡수 증가로 활동에 적합한 상태를 만들고, 밤이나 식사 시에는
부교감신경이 우위가 되어 혈관 확장·맥박 안정·소화 촉진으로 평온한 상태를 만든다.
스트레스란 이 평온한 상태를 깨뜨리는 모든 외부 자극이며, 자극이 오면 교감신경이 작동해
동공 확대, 심박·근육긴장 증가, 말초혈관 수축, 소화기 혈류 감소 등 "전투 준비" 반응이
일어난다. 이 상태가 반복되면 신체가 거기 적응해버리는 것이 만성 스트레스의 기전이다.

### 낮과 밤의 자율신경변화 (idx=313)
자율신경균형(교감/부교감 비율)은 개인차가 있지만 평균 1.1 정도를 중심으로 시소처럼
움직인다. 1.1보다 크면 교감 우위(전투상태), 작으면 부교감 우위(회복상태)다. 낮에는
생체리듬상 자연스럽게 1.1 이상이 되지만, 1.3을 넘으면 과도한 긴장으로 불안·초조와
순환기 문제가 생길 수 있다. 밤에는 1.0 이하로 떨어지며 수면 중엔 0.8까지도 내려가는데,
이때 자가치유가 일어난다. 저녁 9시 이후 취침하지 않고 자정을 넘기면 균형값이 다시
1.1~1.2로 올라가 불면과 다음날 피로로 이어진다. 이 자료는 규림한의원이 이미 확보한
"낮 1.0~1.2 / 밤 0.9~1.1" 정상범위 기준과 정확히 같은 근거를 설명하고 있어, LF/HF
지표 해석 코멘트에 그대로 활용 가능.

### 스트레스가 너무 없어도 생기는 병 (idx=288)
교감신경 긴장으로 인한 질병이 80%를 차지하지만, 나머지 20%는 반대로 부교감신경이
과도하게 우위인 경우다. 부교감 과잉은 혈관이 지나치게 열려 혈액 정체(울혈)와 저체온을
유발하고, 림프구 과잉 반응으로 알레르기·아토피·천식이 생기기 쉬우며, 무기력·과식·비만·
우울감으로 이어질 수 있다. "스트레스가 없는 게 무조건 좋다"는 통념을 교정하는 데
유용한 내용 — 코멘트에서 부교감 과활성 케이스에 참고할 수 있음.

## 2. 스트레스와 신체 질환의 연결고리

### 스트레스와 고혈압 (idx=290)
단기 스트레스에서는 카테콜아민(아드레날린 등)이 혈관을 수축시켜 일시적으로 혈압이
오르지만, 스트레스가 장기화되면 코티솔 등 호르몬 체계 자체가 바뀌면서 혈압 상승이
고착되어 고혈압으로 발전할 수 있다. 스트레스는 고혈압의 여러 위험인자(비만, 운동부족,
흡연, 염분과민, 알코올 등) 중 하나로 언급된다.

### 스트레스와 당뇨 (idx=291)
스트레스 시 코티솔이 혈당을 올리고, 이를 낮추려 인슐린이 과다 분비되는 상태가
지속되면 세포가 인슐린 신호에 둔감해지는 인슐린 저항성이 생겨 당뇨의 초기 기전이 된다.
남는 당은 지방으로 축적되어 비만으로 이어지고, 지방세포의 염증물질이 인슐린 저항성을
더 가속하는 악순환 구조를 설명한다.

### 스트레스와 소화·배설 (idx=285)
교감신경 긴장 시 소화기·장으로 가는 혈류가 약 1/3 감소해 소화불량, 변비, 배뇨장애로
이어질 수 있고, 분비기능 저하로 인한 소화효소 부족과 면역(림프구) 저하도 동반된다는
내용. 만성적으로 지속되면 저체온·면역저하까지 언급되나, 이 부분은 다소 비약이 있어
코멘트에 직접 인용시 "만성적 스트레스가 소화기능에 영향을 줄 수 있다" 정도로 순화 필요.

### 스트레스와 혈액순환 (idx=284)
교감신경 긴장 시 아드레날린이 혈관을 수축시켜 손발이 차가워지고 안색이 창백해지는
현상, 계절(겨울=과립구 증가, 여름=림프구 증가)과 밤낮(밤=부교감 우위, 낮=교감 우위)에
따라 자율신경이 자동 조절되는 원리를 설명. "지속적 긴장·불규칙한 식사·과로·야근·불면"이
이 자동조절을 방해하는 요인으로 명시됨 — 생활관리 코멘트 문구에 활용 가능.

### 스트레스와 면역 관계 (idx=283)
백혈구 중 과립구(세균 담당)와 림프구(바이러스·암세포 담당)의 균형이 교감신경 긴장으로
무너지는 기전을 설명. 과립구 과잉은 활성산소로 인한 조직파괴(궤양, 당뇨 등)로, 림프구
감소는 감염 취약성(잦은 감기 등)으로 이어진다는 내용.

### 약에 의한 스트레스 (idx=289)
진통제, 스테로이드제, 이뇨제 등 흔히 쓰는 약물도 교감신경을 자극하는 방향으로 작용할
수 있다는 관점. 상습적 약 복용이 오히려 교감신경 긴장을 유발할 수 있다는 내용인데,
이 부분은 의학적으로 조심스럽게 다뤄야 할 주제라 — **AI 코멘트에 "약을 줄여야 한다"는
뉘앙스로 직접 반영하지 말 것.** 환자가 복용 중인 약물이 있다면 전문의와 상담하라는
안내 정도로만 순화해서 참고.

## 3. 기기 측정값 해석 관련 (가장 실무 밀접)

### 맥파 측정시 혈관건강도 변화에 대한 해석 (idx=328, 공식 PDF 자료)
심장→대동맥→상완동맥→요골동맥→말초혈관 순으로 혈관이 가늘어지며, 손끝 말초혈관은
동맥의 끝부분이라 혈관 건강상태를 간접적으로 유추하는 데 쓰인다는 원리 설명. 핵심은
**혈관 수축이 혈압·약물·기온·질병 외에 긴장·걱정·피로·불안 같은 정서적 요인으로도
크게 좌우된다**는 점, 그리고 교감신경 긴장 상태는 수초~1분 단위로도 바뀌기 때문에
**같은 사람이라도 측정할 때마다 혈관건강지수가 달라질 수 있다**는 점이다. 이는 "왜
지난 검사와 이번 검사 수치가 다르게 나왔나요?"라는 환자 질문에 대응하는 코멘트
문구에 그대로 활용 가능 — 측정 당시의 컨디션(긴장, 피로, 안정 여부)이 일회성 측정치에
크게 반영된다는 점을 안내하면 신뢰도가 올라감.

### 술과 자율신경 (idx=314, 대표 개인 실측 사례)
음주 후 측정한 자기실험 사례. 소주/양주 각 1회씩, 술은 교감신경은 항진시키고
부교감신경은 저하시키는 방향으로 작용했다는 결과. 장에서 흡수된 알코올이 부교감신경
조절(장 기능)을 방해하는 것으로 추정된다는 내용. **표본 1명, 비공식 자기실험**이므로
"연구에 따르면"이 아니라 "음주 직후 측정 시 자율신경 균형이 일시적으로 흔들릴 수
있어, 검사 전 음주는 피하는 것이 좋다"는 안내 문구 정도로만 활용 권장. (매뉴얼 19p
"음주 및 흡연 후에는 심장박동이 빨라지므로 평상시에 측정" 주의사항과 일치)

### 맛과 자율신경 (idx=317, 방송 실험 인용)
단맛·매운맛 섭취 후 측정한 사례. 매운맛은 통각 자극으로 엔도르핀이 분비되어 교감·
부교감이 함께 오르고 1시간 후 원상복귀, 단맛은 포도당 공급으로 교감이 오르지만 1시간
후에도 남아있어 과다 섭취 시 비만으로 이어질 수 있다는 내용. 두 경우 모두 교감 상승에
따른 일시적 혈관 수축이 공통적으로 관찰됨. 검사 직전 식사/간식 여부를 물어보는 문진
문항 설계에 참고할 만함.

### 공황장애와 자율신경 (idx=315, 대표 가족 사례)
공황장애 증상을 교감신경 과항진 상태로 설명 — 팔다리 근육으로 혈류 집중, 들숨 위주
호흡 전환, 이산화탄소 배출 저하로 인한 호흡곤란까지 이어지는 기전. 약 복용 전후
측정치 비교(교감 6.2→5.6, 자율신경균형 1.2→1.0)를 보여주는 사례. **이 자료는 진단이나
치료 효과를 입증하는 근거가 아니라 개인 사례**이므로, 코멘트에서 공황장애 등 정신과적
진단명을 직접 언급하는 데 사용하지 말 것 — 대신 "가슴 두근거림, 호흡곤란이 반복된다면
전문의 상담이 필요하다"는 안전 안내 수준에서만 참고.

   측정할 때마다 달라질 수 있어, 한 번의 측정보다 여러 번의 추이를 함께 보는 것이
