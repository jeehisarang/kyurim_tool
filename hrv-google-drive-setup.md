# HRV 결과지 구글드라이브 자동가져오기 — 폴더 공유 설정 안내

검사실 노트북에서 HRV 기기가 결과지를 저장하는 구글드라이브 폴더를, 규림툴이 읽어올 수
있도록 아래 절차대로 서비스계정과 공유해주세요. 한 번만 설정하면 이후 계속 자동 연동됩니다.

## 1. 공유할 서비스계정 이메일

```
kyurim-sheet-reader@sage-sylph-501105-f4.iam.gserviceaccount.com
```

(기존 구글폼 설문 연동 때 쓰던 계정과 동일합니다 — 새로 계정을 만들 필요 없습니다.)

## 2. 폴더 공유 절차

1. 구글드라이브(drive.google.com)에서 HRV 결과지가 저장되는 폴더로 이동합니다.
2. 폴더 우클릭 → **공유** (또는 폴더를 열고 우측 상단 "공유" 버튼) 클릭.
3. "사용자 및 그룹 추가"란에 위 서비스계정 이메일을 붙여넣습니다.
4. 권한은 **"뷰어"(읽기 전용)** 로 충분합니다 — "편집자" 권한을 줄 필요 없습니다.
5. "알림 보내기" 체크는 꺼도 됩니다(서비스계정이라 이메일을 확인하지 않음).
6. **완료** 클릭.

## 3. 폴더 ID 확인 방법

1. 구글드라이브에서 해당 폴더를 더블클릭해서 엽니다.
2. 브라우저 주소창의 URL을 확인합니다. 다음과 같은 형태입니다:

   ```
   https://drive.google.com/drive/folders/1a2B3cD4EfGhIjKlMnOpQrStUvWxYz
   ```

3. `folders/` 뒤에 오는 문자열(`1a2B3cD4EfGhIjKlMnOpQrStUvWxYz` 부분)이 폴더 ID입니다.

## 4. 폴더 ID를 저장할 위치

위에서 확인한 폴더 ID를 프로젝트 루트의 `.env` 파일에 아래처럼 추가해주세요(직접 하시기
어려우면 폴더 ID만 알려주시면 대신 넣어드리겠습니다):

```
GOOGLE_HRV_DRIVE_FOLDER_ID=여기에_폴더_ID_붙여넣기
```

## 5. 완료 후 확인

설정이 끝나면 알려주세요 — `/examinations/new`에서 검사종류를 "자율신경맥파기(HRV)"로
선택하고 "구글드라이브에서 가져오기" 버튼을 눌러 폴더 안 파일 목록이 정상적으로 뜨는지
확인하겠습니다.

## 참고 — 지금 당장 결과지를 등록해야 한다면

이 설정이 끝나기 전에도, "구글드라이브에서 가져오기" 버튼 옆의 **"또는 직접 파일 선택"**
으로 컴퓨터에 저장된 결과지 이미지를 바로 업로드해서 등록할 수 있습니다(수동 방식). 폴더
공유는 여러 건을 한 번에 편하게 가져오기 위한 자동화일 뿐, 필수 전제조건은 아닙니다.
