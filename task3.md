# 작업지시서 — HRV 상세지표(TP/VLF/LF/HF/SDNN/RMSSD) 표준범위 조사

## 배경

유비오맥파 리포트("수양명경락기능검사 결과표")에 TP/VLF/LF/HF/자율신경균형
(LF/HF)/평균맥박/SDNN/RMSSD 각각의 "표준범위"와 "OK/--" 판정이 이미 계산되어
표시됨. 실측 샘플(김우석, 44세 남자)에서 평균맥박 표준범위가 "59.5~95.5"로
나왔는데, 이는 기존에 확보한 매뉴얼상의 일반 기준(60~100)과 다름 —
연령/성별에 따라 표준범위 자체가 개인화되어 계산될 가능성이 있음.

이게 사실이라면, CSV로 가져오는 측정치만으로는 OK/-- 판정을 재현할 수 없음
(CSV 컬럼에 "표준범위" 자체는 없고 측정치만 있음 — 지난 조사에서 확인됨).
따라서 이 상세지표들을 코멘트에 활용하기 전에, 표준범위가 어떤 규칙으로
계산되는지부터 확인이 필요함.

## 조사 항목 (구현 없이 조사만)

1. 코드베이스 내 유사 패턴 확인
   - 근력검사(악력)가 이미 "성별·연령대별 기준표(WEAK/NORMAL/STRONG)"를
     쓰고 있다고 알고 있음 — 이 기존 구현이 실제로 어느 파일에 있는지,
     어떤 구조(연령 구간을 어떻게 나눴는지, 데이터를 어디서 확보했는지)로
     되어 있는지 확인. 있다면 이번 HRV 상세지표 표준범위도 같은 구조를
     재사용할 수 있을지 검토
   - exam-thresholds.ts 유사 파일에 TP/VLF/LF/HF/SDNN/RMSSD 관련해서
     이미 남아있는 상수나 주석, 미완성 코드가 있는지 확인 (과거에 시도했던
     흔적이 있을 수 있음)

2. 기기 소프트웨어 설치 파일 내 문서 확인
   - 유비오맥파 프로그램 설치 폴더(예: C:\uBioMacpa Pro\ 등, 실제 설치
     경로는 확인 필요) 안에 도움말 파일(.chm, .pdf, .txt 등)이나 설정
     파일에 연령별 표준범위 표/공식이 포함되어 있는지 확인 — 매뉴얼
     PDF에는 없었지만 설치 폴더 내부에 더 상세한 문서가 있을 가능성 있음

## 이번 조사에서 하지 말아야 할 것

- 표준범위를 임의로 추정해서 코드에 상수로 넣지 말 것 — 근거 없이 넣으면
  나중에 잘못된 정상/비정상 판정으로 이어질 수 있음 (의료 관련 수치라 특히
  주의)
- 실제 UI/AI 코멘트 반영 작업은 착수하지 말 것 — 이번엔 조사만
