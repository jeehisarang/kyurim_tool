# 검사 기록 삭제(비활성화) 기능 추가 — 인바디/근력검사/HRV 공통

## 배경
검사 목록(/examinations) 화면에 개별 기록을 지우는 기능이 없어서, 테스트성/
오등록 데이터를 정리할 방법이 없음. 다른 기능들(Visit, Patient, 프로그램티칭 등)과
동일하게 완전삭제가 아닌 소프트삭제(isActive) 방식으로 추가.

## 요청
1. 세 검사 모델(BodyCompositionRecord, StrengthTestRecord, HrvTestRecord) 모두에
   isActive 필드 확인/추가 (이미 있다면 스킵)
2. 검사 목록(/examinations) 각 행에 삭제 버튼 추가 (또는 상세/편집 화면에)
   - 클릭 시 확인 모달("이 검사 기록을 삭제하시겠습니까?")
   - 확인 시 isActive: false로 소프트삭제 처리
3. 목록 조회 시 기본적으로 isActive: true만 표시 (기존 "/todo" 등에 있던
   "비활성 항목 보기" 토글 패턴 재사용 — 완전히 숨기지 말고 필요시 볼 수 있게)
4. 환자별 검사이력 화면(/examinations/patient/[patientId]), 환자 검사종합
   리포트(/patient-view/exam-report/[patientId]) 등 다른 화면에서 조회할 때도
   비활성 기록이 통계/그래프에 섞여 들어가지 않도록 필터 확인
5. 삭제 권한: 원장 전용으로 할지, 측정자 본인도 가능하게 할지는 기존 패턴
   (Visit 삭제 권한 등) 확인해서 동일하게 맞출 것

## 검증 체크리스트
- 세 검사종류 모두 삭제 버튼 동작 확인
- 삭제 후 목록에서 사라짐, "비활성 항목 보기" 토글 시 다시 보임
- 삭제된 기록이 환자 통합뷰/검사리포트/추이 그래프 등에서 제외되는지 확인
- 기존 정상 기록(비활성화 안 한 것)은 영향 없음 확인
- npx tsc --noEmit, npm run build 통과
- 테스트는 테스트 전용 데이터로 진행 (오늘 쌓인 전태균 HRV 테스트 기록 3건도
  이 기능으로 정리)
