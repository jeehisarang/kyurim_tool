/examinations (검사 목록) 화면 확인 요청:

1. 지금 상단 필터가 "전체보기 / 인바디 / 근력검사" 3개뿐이고, 자율신경맥파(HRV)
   탭이 없음. "전체보기"에도 HRV 기록이 안 보이는데, 이게 필터/쿼리에서 HRV를
   아예 빼고 있는 건지, 아니면 정말 저장이 안 되고 있는 건지 먼저 확인해줘
   (DB에서 HrvTestRecord 테이블에 오늘 등록한 기록들이 실제로 있는지 직접 조회)

2. DB에 정상 저장되어 있다면: 이 목록 화면에 HRV 탭을 추가하고, HRV 전용
   컬럼(혈관건강지수/혈관건강도/평균맥박/스트레스지수 등)이 "전체보기"에서는
   인바디/근력검사와 다른 형태이니 어떻게 표시할지 방식을 제안해줘
   (예: 전체보기에서는 검사종류별로 컬럼이 다르게 나오는 게 자연스러움, 별도
   요약 컬럼 하나로 축약하는 방법도 있음 — 기존 코드 구조에 맞는 쪽으로 판단)

3. 클릭 시 이동 경로도 확인: 인바디/근력검사처럼 원장 확인 화면
   (/examinations/hrv/[id])으로 정상 연결되는지
